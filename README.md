# Pygame Pathfinding Visualizer

Shows a visualization of Pathfinding Algorithms from a source node to a target node. Create your own walls and try it out!

## Color Dictionary

| Color | Meaning |
|-------|---------|
| White |  Empty Node  |
| Black |  Wall Node  |
| Green |  Source Node  |
| Red |  Target Node  |
| Dark Blue |  Searched Node  |
| Light Blue |  Discovered Node  |

White       - Empty Square.
Black       - Wall.
Green       - Source.
Red         - Target.
Dark Blue   - Searched Node.
Light Blue  - Discovered Node.

## Controls

|  Button |     Action   |
|---------|--------------|
|    S    | Place Source |
|    T    | Place Target |
|    R    |      Reset   |
|    1    |    Run BFS   |
|   Click |  Place Wall  |
